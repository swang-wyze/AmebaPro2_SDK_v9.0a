Example Description

This example describes how to use i2c by using mbed api

1.Connect LOG-UART connector to PC
  
2.Connect board 1 and board 2
       board 1            board 2
  - I2C0 SDA (PF_3) to I2C3 SDA (PF_3) pin and also connect to a pull-up resistor,
  - I2C0 SCL (PF_2) to I2C3 SCL (PF_2) pin and also connect to a pull-up resistor.
  
3.Run the main function.

4.Get the Master and Slave Data.
